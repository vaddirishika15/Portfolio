Language Translation

Developed a language translation project using a tech stack that includes Keras for machine learning models, NumPy and Pandas for data manipulation, and Python as the primary programming language. This project focuses on efficiently translating text between German to English and English to German languages. Leveraging Keras, I implemented machine learning models to ensure accurate and contextually relevant translations. NumPy and Pandas handle data preprocessing and management tasks, ensuring efficient handling of linguistic data for both translation directions.

Movie Recommendation System

Developed a movie recommendation system using Python, Pandas, Scikit-learn, and Pickle, leveraging machine learning techniques for data processing and analysis. The system is designed to suggest movies based on user-selected preferences, enhancing personalized recommendations. The project utilizes a dataset comprising 5000 movies collected from Kaggle, encompassing various attributes such as genre, ratings, and reviews. Using Pandas, I performed extensive data preprocessing to clean and organize the dataset for effective analysis. Scikit-learn's machine learning algorithms were employed to build recommendation models, which were serialized using Pickle for efficient storage and retrieval. This project not only demonstrates proficiency in data handling and machine learning but also showcases practical applications of recommendation systems in enhancing user experience
